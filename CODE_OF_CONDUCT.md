# Code of Conduct

## Our Pledge

This is a personal chess engine project by James. Everyone who interacts with this repository — whether opening issues, submitting pull requests, or leaving comments — is expected to be respectful and constructive.

## Expected Behaviour

- Be kind and respectful in all interactions
- Give constructive feedback, not personal criticism
- Stay on topic — this is a chess engine project
- Credit others' ideas and contributions appropriately

## Unacceptable Behaviour

- Harassment, insults, or personal attacks of any kind
- Spam or off-topic promotion
- Deliberately unhelpful or destructive contributions

## Scope

This code of conduct applies to all project spaces — GitHub issues, pull requests, discussions, and any other communication related to this project.

## Enforcement

As this is a personal project, James reserves the right to remove comments, close issues, or block users who violate this code of conduct.
